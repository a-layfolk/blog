# Lab5 语法分析实验

# 1. 实验目的和内容

实验目的：

  • 熟悉 C 语言的语法规则，了解编译器语法分析器的主要功能
  • 熟练掌握典型语法分析器构造的相关技术和方法，设计并实现具有一定复杂度和分析能力的 C 语言语法分析器
  • 了解 ANTLR 的工作原理和基本思想，学习使用工具自动生成语法分析器
  • 掌握编译器从前端到后端各个模块的工作原理，语法分析模块与其他模块之间的交互过程

实验内容：

  • 选择 C 语言的一个子集，基于 BIT-MiniCC 构建 C 语言子集的语法分析器
  • 该语法分析器能够在输入正确时输出 JSON 格式的语法树

# 2. 实验具体过程和步骤

实验具体步骤：

  • 参照给出的文法，扩充定义自己希望实现的 C 语言语法子集
  • 使用 ANTLR 自动生成语法分析器
  • 将语法树输出为 JSON 文件

# 2.1.1 声明语句

出于简单起见，给出的声明中，并没有包括数组、结构体、枚举、联合和指针

/* 语法分析 */
// 声明语句 , 包括变量声明语句 和 函数声明语句
declaration     :   var_declaration
                |   fun_declaration
                ;

// 变量声明语句 , 例如 int a;
var_declaration :   type_specifier IDENTIFIER ';';
// 函数声明语句 , 例如 int sum(int a, int b);
fun_declaration :   type_specifier IDENTIFIER '(' params ')' ';';
// 类型修饰符
type_specifier  :   'char'
                |   'double'
                |   'float'
                |   'int'
                |   'long'
                |   'short'
                |   'signed'
                |   'unsigned'
                |   'void'
                ;
// 函数参数 , 包括有参数和 void
params  :   params_list
        |   'void'
        ;
// 有参数的情 , 包括只有一个参数 和 含有多个参数
params_list :   type_specifier IDENTIFIER
            |   params_list ',' type_specifier IDENTIFIER
            ;

/* 词法分析 */
IDENTIFIER  :   [a-zA-Z_][a-zA-Z0-9_]*;
WS	        :   [ \t\r\n]+ -> skip;

# 2.1.2 表达式语句

直接给出赋值语句的文法比较困难，因为赋值语句包括多种情况，所以我们先给出表达式语句，再根据表达式语句给出赋值语句

/* 语法分析 */
// 表达式语句 , 包括算术运算 和 比较运算
expression_stmt :   expression ';';
expression  :   arithmetic_expression
            |   arithmetic_expression com_op arithmetic_expression
            ;
// 算术表达式语句 ， 包括加减运算 和 乘除运算
arithmetic_expression   :   add_sub_expression
                        |   mul_div_expression
                        ;
// 加减运算
// 为防止 加减运算 和 乘除运算相互左递归
// 乘除运算 、 加减运算均有时
// 归于 加减运算
add_sub_expression  :   factor
                    |   factor add_sub_op add_sub_expression
                    |   add_sub_expression add_sub_op factor
                    |   factor add_sub_op mul_div_expression
                    |   mul_div_expression add_sub_op factor
                    ;
// 乘除运算
mul_div_expression  :   factor
                    |  factor mul_div_op mul_div_expression
                    |  mul_div_expression mul_div_op factor
                    ;
// 运算时的每一项 , 可能时整数、浮点数、变量、表达式
factor  :   integer_constant
        |   floating_constant
        |   var
        |   '('expression')'
        ;
add_sub_op  :   '+'
            |   '-'
            ;
mul_div_op  :   '*'
            |   '/'
            ;
com_op  :   '<='
        |   '<'
        |   '>'
        |   '>='
        |   '=='
        |   '!='
        ;
integer_constant    :   INTEGER_CONSTANT;
floating_constant   :   FLOATING_CONSTANT;
var :   IDENTIFIER;
/* 词法分析 */
// 为简单起见 , 整数只包括十进制整数 , 没有前缀和后缀
INTEGER_CONSTANT    :   [1-9][0-9]*;
// 为简单器件 , 浮点数也只包括十进制浮点数 , 没有前缀和后缀
FLOATING_CONSTANT   :   [0]+[0-9]*'.'[0-9]*;
IDENTIFIER  :   [a-zA-Z_][a-zA-Z0-9_]*;
