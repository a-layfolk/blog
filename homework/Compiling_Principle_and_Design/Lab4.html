# Lab4 文法设计实验

# 1. 实验目的和内容

实验目的:

  • 了解程序设计语言的演化
  • 了解相关标准的制定过程
  • 深入理解与编译实现的有关形式语言理论
  • 使用文法对给定语言进行描述
  • 为词法分析和语法分析做准备

实验内容:

  • 阅读提供的 C 语言和 Java 语言规范草稿，了解语言规范化定义应包括的具体内容

  • 选定 C 语言子集，并使用 BNF 表示方法文法进行描述，要求至少包括表达式、赋值语句、分支语句和循环语句；或者设计一个新的程序设计语言，并使用文法对该语言的词法规则和语法规则进行描述。

    以上语言定义首先要给出所使用的字母表，在此基础上使用 2 型文法描述语法规则。

# 2. 语言规范理解

在附件 C11-n1570 xvii 页的 Introduction 中，有：

This International Standard is divided into four major subdivisions:
— preliminary elements (clauses 1−4);
— the characteristics of environments that translate and execute C programs (clause 5);
— the language syntax, constraints, and semantics (clause 6);
— the library facilities (clause 7).		 此国际标准分为四个主要部分：
— 初步要素（第 1-4 条）
— 翻译和执行 C 语言程序的环境特性（第 5 条）
— 语言语法、约束和语言（第 6 条）
— 库设置（第 7 条）

而在附件 jls12.pdf 的 Introduction 中，有

Chapter 3 describes the lexical structure of the Java programming language, which is based on C and C++.

......		 第 2 章描述了语法，表示语言语法和句法语法的符号。

# 3.1 单词表

首先给出单词表，这里给出的单词表在 Lab3 中使用过。左边会给出单词表，右边会着重给出之后 BNF 分析时要用到的单词。

# 3.2 BNF 描述文法

现在给出 BNF 描述文法，首先给出整体的轮廓，然后层层下推。

statement ::= expression-stmt
              | assignment-stmt
              | selection-stmt
              | iteration-stmt
              | transfer-stmt
              | label-stmt

C 语言中的语句包括：表达式语句、赋值语句、分支语句、循环语句、跳转语句和标记语句。

这里我把 switch case 语句中的 case 和 default 语句称为标记语句。

此外，和语句有些差别的是声明，比如变量声明和函数声明。

# 3.2.1 表达式语句

  • 1 表达式语句由表达式后接分号组成

    expression-stmt ::= expression ;

  • 1 表达式 expression 一般就是算术运算和比较运算

    expression ::= arithmetic-expression
               | arithmetic-expression com-op arithmetic-expression

  • 1-1 算术运算包括加减运算和乘除运算

    arithmetic-expression ::= add-sub-expression
                          | mul-div-expression

  • 1-1-1 其中，加减运算中的某一项或几项可能是乘除运算

    add-sub-expression ::= factor
                       | add-sub-expression add-sub-op add-sub-expression
                       | add-sub-expression add-sub-op mul-div-expression
                       | mul-div-expression add-sub-op add-sub-expression
                       | mul-div-expression add-sub-op mul-div-expression

  • 1-1-1-1 factor 表示参与运算的元素，包括整型常量、浮点数型常量、变量、表达式。其中整型常量、浮点数型常量在字母表中已经定义

    factor ::= integer-constant
           | floating-constant
           | var
           | ( expression )

  • 1-1-1-1-1 变量则可以是标识符或者数组，其中 identifier 和 decimal-constant 在字母表中已经定义

    var ::= identifier
        | identifier [ decimal-constant ]

  • 1-1-1-2 add-sub-op 则就是简单的 + 和 -

    add-sub-op ::= + | -

  • 1-1-2 乘除运算中的某一项也可能是加减运算

    mul-div-expression ::= factor
                     | add-sub-expression mul-div-op add-sub-expression
                     | add-sub-expression mul-div-op mul-div-expression
                     | mul-div-expression mul-div-op add-sub-expression
                     | mul-div-expression mul-div-op mul-div-expression

  • 1-1-2-1 factor 之前已经定义过，mul-div-op 就是简单的 * 和 /

    mul-div-op ::= * | /

  • 2 com-op 比较运算符

    com-op ::= <=
           | <
           | >
           | >=
           | ==
           | !=

# 3.2.2 赋值语句

  • 赋值语句中，var 和 expression 之前已经定义过了，主要看 ass-op

    assignment-stmt ::= var ass-op expression ;

    ass-op ::= =
           | +=
           | -=
           | *=
           | /=
           | %=
           | <<=
           | >>=
           | &=
           | |=
           | ^=

# 3.2.3 分支语句

  • 分支语句包括 if，if else 和 switch 分支

    selection-stmt ::= if ( expression ) { statement }
                   | if ( expression ) { statement } else { statement }
                   | switch ( expression ) { statement }

# 3.2.4 循环语句

  • 循环语句包括 while，do while 和 for 循环

    iteration-stmt ::= while ( expression ) { statement }
                   | do { statement } while ( expression ) ;
                   | iteration-for

    iteration-for ::= for ( expression ; expression ; expression ) { statement }

# 3.2.5 跳转语句

  • 跳转语句包括 goto，break，continue 和 return 语句

    transfer-stmt ::= goto identifier ;
                  | continue ;
                  | break ;
                  | return-stmt

    返回语句要么为 return ; （函数返回值为空时）要么为 return expression

    return-stmt ::= return ;
                | return expression ;

# 3.2.6 标记语句

  • 标记语句主要就包括两个，一个 case，一个 default

    lable-stmt ::= case integer-constant : statement
               | case floating-constant : statement
               | case character-constant : statement
               | default : statement

# 3.2.7 声明

  • 1 声明包括变量声明和函数声明

    declaration ::= var-declaration
                | fun-declaration

  • 1-1 变量声明比较简单

    var-declaration ::= type-specifier identifier ;
                    | type-specifier identifier [ decimal-constant ]

  • 1-1-1 其中，变量标识符是许多个关键字

    type-specifier ::= char
                   |double
                   |float
                   |int
                   |long
                   |short
                   |signed
                   |unsigned
                   |void

  • 1-2 函数声明会包括变量标识符，identifier，还有参数，以及函数体

    fun-declaration ::= type-specifier identifier ( params );

  • 1-2-1 参数的定义比较复杂，有可能有一个、多个参数，也可能为空

    params ::= param-list
           | void

    params-list ::= param
                | params-list , param

    参数不仅可能是变量，也可能是数组

    param ::= type-specifier identifier
          | type-specifier identifier [ ]

# 4. 实验心得体会

我认为本次实验，完成比较简单，但是想要完整的标识却比较困难。比如我给出的变量标识符为：

type-specifier ::= char
                   |double
                   |float
                   |int
                   |long
                   |short
                   |signed
                   |unsigned
                   |void

但事实上，C 语言中变量标识符还有 struct，union，enum 和 typedef 等。

如果这些标识符用于声明 C 语言的变量，真正需要完整考虑的话，还应该注意 const 和 volatile 修饰符。此外，虽然变量我已经将数组考虑了进去，但并没有将指针考虑进去，且如果真正深入考虑的话，除了一维数组还有多维数组等需要考虑。
