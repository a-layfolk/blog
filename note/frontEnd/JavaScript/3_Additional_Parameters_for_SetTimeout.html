# setTimeout 的附加参数

有关 setTimeout 的用法， MDN (opens new window) 中已经有详细解释。 这段文字主要讨论 setTimeout 附加参数的问题。

setTimeout(function[, delay, arg1, arg2, ...]);

一旦定时器到期， 附加参数会作为参数传递给 function

举例如下：

 setTimeout(function (arg1) {
   console.log(arg1);
 }, 100, "one");
// 打印 one

 setTimeout(function (arg1, arg2, arg3) {
   console.log(arg1, arg2, arg3);
 }, 100, "one", "two", "three");
 // 打印 one， two， three

TIP

需要注意 setTimeout 与 Promise 结合的情况， Promise 的 resolve 函数只接收一个参数。 因此想要传递多个参数时， 应当把多个参数作为数组。

举例如下：

function timeout(ms) {
  return new Promise(function(resolve, reject) {
    setTimeout(resolve, ms, ["one", "two", "three"]);
  });
}

timeout(100).then(function(value) {
  console.log(value);
});
// 打印 ["one", "two", "three"]
