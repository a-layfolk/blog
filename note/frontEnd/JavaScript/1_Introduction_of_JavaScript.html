# JavaScript 简介

JavaScript 最初设计是为完成客户端的基本验证任务。现如今已成长为一门强大的编程语言。

JavaScript 包括三部分：

  • 核心（ECMAScript）
  • BOM（Browser Object Model，浏览器对象模型）
  • DOM（Document Object Model，文档对象模型）

TIP

BOM 一开始没有标准可以遵循，直到 HTML5 才得到解决。

TIP

DOM 标准是由 Web 通信标准制定者 W3C（World Wide Web Consortium，万维网联盟）规划的，除了 JavaScript，还有其他语言实现了 DOM，比如 SVG 就是 XML 语言实现的。
