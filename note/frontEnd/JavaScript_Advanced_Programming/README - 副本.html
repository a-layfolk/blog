# 操作元素

# 4.1 改变元素内容

element.innerText

从起始位置到终止位置的位置，但它去除 html 标签，同时空格和换行也会去掉

element.innerHTML

起始位置到终止位置的全部内容，包括 html 标签，同时保留空格和换行。

# 4.2 常用元素的属性操作

  1. innerText、innerHTML 改变元素内容

  2. src、href

    var img = document.querySelector("img");
img.src = "images/newImg.jpg";

  3. id、alt、title

# 4.3 表单元素的属性操作

利用 DOM 可以操作如下表单元素的属性：

type、value、checked、selected、disabled

var input = document.querySelector("input");
input.value = "要修改表单的文字内容，必须使用 value";

var btn = document.querySelector("button");
btn.onclick = function() {
    this.disabled = true;	// this 指向的是事件函数的调用者
}

点击按钮，将 input 的属性从 password 变为 text

# 4.4 样式属性操作

我们可以通过 JS 修改元素的大小、颜色、位置等样式

1. element.style			行内样式操作
2. element.className		类名样式操作

div.onclick = function() {
    // div.style 里面的属性采取驼峰命令法
	this.style.backgroundColor
}
  • JS 里面的样式采取驼峰命名法，比如 fontSize、backgroundColor
  • JS 修改 style 样式操作，产生的是行内样式，css 权重比较高

循环精灵图案例：

  • 首先精灵图图片排列是有规律的
  • 核心思路，利用 for 循环，修改精灵图片的北京位置 background-position
var lis = document.querySelectorAll("li");
for (var i = 0; i < lis.length; i++) {
    var index = i * 44;
    lis[i].style.backgroundPosition = "0 -" + index + "px";
}

<style>
    .change {
        ...
    }
</style>
<body>
    <script>
      var test = document,querySelector("div");
      // 我们可以修改 元素的 className 更爱元素的样式
      test.onclick = function() {
          this.className = "change";
      }
    </script>
</body>

注意：

  • 如果样式修改较多，可以采取操作类名方式更改元素样式
  • class 因为是个保留字，因此使用 className 类操作元素类名属性
  • className 会直接更改元素类名，会覆盖原有的类名

如果想要保留原先的类名，我们可以这样写

test.onclick = function() {
    this.className = "first change";
}

# 4.5 排他思想

如果有同一组元素，我们想要某一个元素实现某种样式，需要用到循环的排他算法：

  • 所有元素全部清除样式（干掉其他人）

  • 给当前元素设置样式（留下我自己）
