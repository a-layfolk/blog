# 引用类型

# Object 类型

创建 Object 实例有两种方式：

  • 使用 new 操作符后跟 Object 构造函数

    var person = new Object();
person.name = "Nicholas";
person.age = 29;

  • 使用对象字面量

    var person = {
    name: "Nicholas",
    age: 29
};

    TIP

    在通过对象字面量定义对象时，实际上不会调用 Object 构造函数。

alert(person["name"]);
alert(person.name);

// 使用方括号的主要优点是可以使用变量来访问属性
var propertyName = "name";
alert(person[propertyName]);	// "Nicholas"

// 如果属性名中包含可能会导致语法错误的字符，可以使用方括号
person["first name"] = "Nicholas";

# Array 类型

ES 数组的每一项可以保存任何类型的数据

ES 数组的大小可以动态调整

创建数组的基本方式有两种：

  • 使用 Array 构造函数

    var colors = new Array();

    // 也可以直接传递长度
var colors = new Array(20);

    // 也可以直接包含项
var colors = new Array("red", "green", "blue");

  • 使用数组字面量

    var colors = ["red", "green", "blue"];

方括号中的索引表示要访问的值。

数组的项数保存在其 length 属性中。

# 检测数组

  • 方法一：instanceof

    if (value instanceof Array) {
    // 对数组执行的擦欧总
}

    这个方法问题在于，它假定单一的全局执行环境。如果网页有包含多个框架，就存在两个以上不同的 Array 构造函数。

  • 方法二：ES5 的 Array.isArray() 方法

    if (Array.isArray(value)) {
    // 对数组执行某些操作
}

# 转换方法

所有对象都具有 toLocalString()，toString() 和 valueOf 方法

# 栈方法

  • push() 接受任意数量的参数，把它们诸葛添加到数组末尾
  • pop() 从数组末尾移除最后一项，减少数组的 length 值，然后返回移除的项

# 队列方法

  • shift() 取出数组的第一项，和 push() 一起可以模拟队列
  • unshift() 方法从数组前端添加任意个项，和 pop() 可以从相反方向模拟队列

# 重排序方法

  • reverse()，反转数组项的顺序

  • 默认情况下，sort() 方法会按升序排列数组

    TIP

    sort() 方法会调用每个数组项的 toString() 转型方法，然后比较得到的字符串

    很多时候，为了比较数组，我们需要给 sort() 函数传递一个比较函数作为参数。

    function compare(value1, value2) {
    if (value1 < value2) {
		return -1;
    } else if (value1 > value2) {
        return 1;
    } else {
        return 0;
    }
}

var values = [0, 1, 5, 10, 15];
values.sort(compare);
alert(values);	// 0, 1, 5, 10, 15

    function compare(value1, value2) {
    if (value1 < value2) {
		return 1;
    } else if (value1 > value2) {
        return -1;
    } else {
        return 0;
    }
}

var values = [0, 1, 5, 10, 15];
values.sort(compare);
alert(values);	// 15, 10, 5, 1, 0

    对于数值类型或者其 valueOf() 方法可以返回数值类型的对象来说，可以使用一个更简单的函数：

    function compare(value1, value2) {
    return value2 - value1;
}

# 操作方法

  • concat() 方法

    • 先创建一个当前数组的副本
    • 接受到的参数添加到数组末尾
    var colors = ["red", "green", "blue"];
var colors2 = colors.concat("yellow", ["black", "brown"]);

alert(colors);		// red, green, blue
alert(colors2);		// red, green, blue, yellow, black, brown

  • slice() 方法

    • 接受一或两个参数，即要返回项的其实和结束位置
    • slice() 方法不会影响原始数组
    var colors = ["red", "green", "blue", "yellow", "purple"];
var colors2 = colors.slice(1);
var colors2 = colors.slice(1,4);

alert(colors2);		// green, blue, yellow, purple
alert(colors3);		// green, blue, yellow

  • splice() 方法，可以说是最强大的数组方法

    • 删除：可以删除任意数量的项，只需指定两个参数：要删除的第一项位置和要删除的项数。例如 splice(0, 2) 会删除数组中的前两项

    • 插入：可以向指定位置插入任意数量的项，只需提供单个参数，起始位置、0（要删除的项数）和要插入的项。如果要插入多个项，可以再传入第四、第五、乃至任意多个项。

      例如 splice(2, 0, "red", "green") 会从当前数组的位置 2 开始插入字符串 “red" 和 ”green"

    • 替换，可以项指定位置插入任意数量的项，且同时删除任意数量的项，插入的项数不必与删除的项数相等。

    • splice() 方法始终都会返回一个数组，该数组中包含从原始数组中删除的项（如果没有删除任何项，则返回一个空数组）

# 位置方法

indexOf() 和 lastIndexOf()

两个参数：

  • 要查找的项
  • （可选）表示查找起点位置的索引
var numbers = [1, 2, 3, 4, 5, 4, 3, 2, 1];

alert(numbers.indexOf(4));			// 3
alert(numbers.lastIndexOf(4));		// 5

alert(numbers.indexOf(4, 4));		// 5
alert(numbers.lastIndexOf(4, 4));	// 3

# 迭代方法

ES5 为数组定义了 5 个迭代方法，每个方法接受两个参数：

  • 要在每一项上运行的函数

    函数会接收 3 个参数：

    • 数组项的值
    • 该项在数组项中的位置
    • 数组对象本身

  • （可选）运行该函数的作用域对象——影响 this 的值

这五个迭代方法如下：

every()

对数组中的每一项运行给定函数，如果该函数对每一项都返回 true，则返回 true

var numbers = [1, 2, 3, 4, 5, 4, 3, 2, 1];

var everyResult = numbers.every(function(item, index, array) {
    return (item > 2);
});

alert(everyResult);		// false

filter()

对数组中的每一项运行给定函数，返回该函数会返回 true 的项或者数组

var numbers = [1, 2, 3, 4, 5, 4, 3, 2, 1];

var filterResult = numbers.filter(function(item, index, array) {
    return (item > 2);
});

alert(filterResult);	// [3, 4, 5, 4, 3]

forEach()

对数组中的每一项运行给定函数，这个函数没有返回值。

var numbers = [1, 2, 3, 4, 5, 4, 3, 2, 1];

numbers.forEach(function(item, index, array) {
    // 执行某些操作
});

map()

对数组中的每一项运行给定函数，返回每次函数调用的结果组成的数组

var numbers = [1, 2, 3, 4, 5, 4, 3, 2, 1];

var mapResult = numbers.map(function(item, index, array) {
   return item * 2; 
});

alert(mapResult);	// [2, 4, 6, 8, 10, 8, 6, 4, 2]

some()

对数组中每一项运行给定函数，如果该函数对任一项返回 true，则返回 true

var numbers = [1, 2, 3, 4, 5, 4, 3, 2, 1];

var someResult = numbers.some(function(item, index, array) {
    return (item > 2);
});

alert(someResult);	// true

以上方法都不会修改数组中包含的值

# 缩小方法

ES5 中新增了两个缩小数组的方法：reduce() 和 reduceRight()。这两个方法都会迭代数组的所有项，然后构建一个最终返回的结果值。

reduce() 方法从数组第一项开始，reduceRight() 从数组最后一项开始。

两个方法都接收两个参数：

  • 一个在每一项上调用的函数

    函数会接收 4 个参数

    • 前一个值
    • 当前值
    • 项的索引
    • 数组对象

  • （可选）作为缩小基础的初始值

函数返回的任何值都会作为第一个参数自动传给下一项。

var values = [1, 2, 3, 4, 5];

var sum = values.reduce(prev, cur, index, array {
	return prev + cur;                        
});
alert(sum);	// 15
  • 第一次执行回调函数，prev 是 1 ，cur 是 2
  • 第二次执行回调函数，prev 是 3（1 + 2 的结果），cur = 3

reduceRight() 作用与之类似，只不过方向相反。

# Date 类型

Date 类型使用自 UTC (Coordinated Universal Time，国际协调时间) 1970 年 1 月 1 日午夜（零时）开始经过的毫秒数来保存日期。

要创建一个日期对象，使用 new 操作符和 Date 构造函数即可。

var now = new Date();	// 新对象自动获取当前的日期和时间

如果要想根据特定的日期和时间创建日期对象，必须传入该日期的毫秒数。

为了简化这个过程，ES 提供了 Date.parse() 和 Date.UTC() 两个方法。

  • Date.parse() 接受一个表示日期的字符串参数

    这个字符串并没有规定格式，如果浏览器地区设置为美国，通常接收

    • “月/年/日”，如 6/13/2004
    • "英文月名 日,年"，如 January 12,2004
    • “英文星期几 英文月名 日 年 时:分:秒:时区”，如 Tue May 25 2004 00:00:00 GMT-0700
    • ISO 8601 拓展格式 YYYY-MM-DDTHH:mm:ss.sssZ（例如 2004-05-25T00:00:00）
    var someDate = new Date(Date.parse("May 25, 2004"));

// 如果没有 Date.parse()，后台也会调用
var someDate2 = new Date("May 25, 2004");

  • Date.UTC() 方法的参数分别是年份、基于 0 的月份（一月是 0，二月是 1，依次类推），月中的一天（1-31）、小时数（0~23），分钟、秒以及毫秒数。

    • 其中只有年和月必须
    // GMT 时间 2000 年 1 月 1 日 午夜零时
var y2k = new Date(Date.UTC(2000,0));

// GMT 时间 2005 年 5 月 5 日下午 5:55:55
var allFives = new Date(Date.UTC(2005,4,5,17,55,55))

    如同模仿 Date.parse() 一样，Date 构造函数也会模仿 Date.UTC()

    但有一点不同，日期和时间都是基于本地时区而非 GMT 来创建

  • Date.now() 是 ES5 新增的，返回表示调用这个方法时的日期和毫秒数

# 继承的方法

Date 类型重写了 toLocalString()、toString() 和 valueOf() 方法：

  • toLocalString() 会按照浏览器设置的地区相适应的格式返回日期和时间

    一般不会包含时区，具体情况看浏览器

  • toString() 通常会包含时区，一般用军用时间

  • valueOf() 返回日期的毫秒表示

# RegExp 类型

var expression = / pattern / flags;
  • g：表示全局模式，模式将被应用于所有字符串。
  • i：表示不区分大小写模式
  • m：表示多行模式

或者：

var pattern = new RegExp("[bc]at", "i");

# RegExp 实例属性

  • global：布尔值，表示是否设置了 g 标志
  • ignoreCase：布尔值，表示是否设置了 i 标志
  • lastIndex：整数，表示开始搜索下一个匹配项的字符位置，从 0 算起
  • multiline：布尔值，表示是否设置了 m 标志
  • source：正则表达式的字符串表示，按字面量形式

# RegExp 实例方法

RegExp 对象的主要方法时 exec()。

exec() 接收一个参数，返回包含第一个匹配项信息的数组；

数组中有两个额外的属性：index 和 input

  • index 表示匹配项在字符串中的位置
  • input 表示应用正则表达式的字符串

对于 exec() 方法而言，即使在模式中设置了全局标志（g），它每次也只会返回一个匹配项。

  • 不设置全局标志时，exec() 始终返回第一个匹配项的信息
  • 设置全局标志时，exec() 每次都会在字符串中继续查找新匹配项
