# 列表渲染

微信小程序的列表渲染语法和 Vue3 的列表渲染语法稍有不同。

微信小程序可以在标签上直接使用 mustache 语法， Vue3 不可以。

  • 微信小程序的例子 (opens new window)
    <view wx:for="{{array}}">
  {{index}}: {{item.message}}
</view>

    Page({
  data: {
    array: [{ message: "foo" }, { message: "bar" }]
  }
})
  • Vue3 的例子 (opens new window)
    <ul id="array-rendering">
  <li v-for="item in items">
    {{ item.message }}
  </li>
</ul>

    Vue.createApp({
  data() {
    return {
      items: [{ message: "Foo" }, { message: "Bar" }]
    }
  }
}).mount('#array-rendering')
