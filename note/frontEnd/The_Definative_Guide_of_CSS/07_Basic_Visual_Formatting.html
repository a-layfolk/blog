# 基本视觉格式化

# 基本框

CSS 假定每个元素都会生成一个或多个矩形框，称为元素框。元素框包括内边距、边框和外边距。

  • 内容的背景（例如某种颜色或平铺图像）也会应用到内边距。
  • 外边距通常是透明的。
  • 内边距不能是负值。
  • 外边距可以是负值。

# 块级元素

  • 一般地，一个元素的 width 被定义为从左内边界到右内边界的距离，height 则是从上内边界到下内边界的的距离。

# 水平格式化

<p style="width: 200px; padding: 10px; margin: 20px;">
    wideness?
</p>
  • 可见元素框的宽度是 220px，包括左右的 10px 像素。
  • 整个元素框的宽度是 260px，是从左外边界到右外边界

水平属性

水平格式化的“七大属性”：margin-left，border-left，padding-left，width，padding-right，border-right 和 marigin-right。

这 7 个属性的值加在一起必须是元素包含块的宽度，也往往是父元素的 width 值。

只有三个属性可以设置为 auto:

  • 元素内容的 width

  • 左、右外边距

  • 如果设置 width，margin-left 或 margin-right 中某个值为 auto，而余下两个属性指定为特定的值，那么设置为 auto 的属性会确定所需的长度。

  • 如果没有任何一个属性设置为 auto，而元素总宽度又不等于包含块的 width，会强制将 margin-right 设置为 auto

  • 如果两个外边框都显示设置，width 设置为 auto，会将 width 变化到总长度为父元素的内容宽度

  • 如果左右外边距都设置为 auto，元素居中

  • 某个外边距和 width 设置为 auto，设置为 auto 的外边距变为 0

  • 三个属性均设置为 auto，则两个外边距都变为 0

TIP

负外边距：

设置 margin-right 为负值，width 为 auto，此时计算之后，子元素的宽度会超过父元素，但是 7 个属性的属性值加在一起仍等于所需的总宽度。

设置 margin-left 为负值，不仅子元素会超过父元素的边框，还可能超出浏览器窗口本身的边界。

百分数

width、内边距和外边距可以设置为百分数，也会应用之前的规则。它们的百分数是相对于包含块的宽度的。

但是边框不能是百分数，只能是长度。所以基本原则是：只使用百分数无法创建完全灵活的元素布局，除非不想使用边框。

替换元素

非替换元素的所有规则同样使用于替换块元素，只有一个例外，如果 width 为 auto，元素的宽度则是内容的固有宽度。

# 垂直格式化

垂直格式化也有 7 个相关的属性：margin-top，border-top，padding-top，height，padding-bottom，border-bottom 和 margin-bottom。

这七个属性的值必须等于元素包含块的 height。

这七个属性中只有 height 和上、下外边距可以设置为 auto

  • 如果一个正常块中的 margin-top 或 margin-bottom 设置为 auto，它会自动计算 0

TIP

嵌套块元素垂直外边距的塌陷。

对于两个嵌套关系（父子关系）的块元素，父元素有上外边距同时子元素也有上外边距，此时父元素会塌陷到较大的外边距。

解决办法：

  • 可以给父元素定义边框
  • 可以给父元素定义内边距
  • 可以为父元素添加 overflow: hidden

合并垂直外边距

垂直格式化的另一个重要方面是垂直相邻外边距的合并，这种合并行为只应用于外边距。如果元素有内边距和边框，它们绝对不会合并。

  • 相邻元素，两个外边距中较小的一个会被较大的一个合并

负外边距

负外边距会影响外边距如何合并：

  • 如果垂直外边距都设置为负值，则浏览器会取绝对值的最大值
  • 如果一个正外边距与一个负外边距合并，会从正外边距减去整个负外边距的绝对值

# 行内元素

  • 相邻行内元素在一行上，一行可以显示多个
  • 高、宽直接设置时无效的
  • 默认宽度就是它本身的宽度
  • 行内元素只能容纳文本或其他行内元素

TIP

  • 连接里面不能再放链接
  • 特殊情况链接 <a> 里面可以放块级元素，但是给 <a> 转换以下块级模式最安全

如果想让行内元素或行内块元素水平居中对齐，给其父元素添加 text-align: center

# 行内块元素

这并不是官方的说法

<img /> <input /> <td>

它们同时具有行内元素和块级元素的特点。

  • 和相邻行内元素在一行上，但是他们呢之间会有空白间隙，一行可以显示多个（行内元素特点）
  • 默认宽度就是它本身内容的宽度（行内元素特点）
  • 宽度、行高、外边距以及内边距都可以控制
