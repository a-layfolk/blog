# 字体

设置字体属性是样式表最常用用途之一。

尽管 CSS2 支持可下载字体，并定义了相应的属性，不过处于性能方面的原因，阅读器总会拒绝下载字体。

CSS 定义了 5 种通用字体系列

  • Serif 字体

    字体成比例，而且有上下短线

  • Sans-serif 字体

    成比例，没有上下短线，此字体系列的字体包和 Helvetica、Verdana 和 Arial 等

  • Monospace

    模拟打印机打出的文本，字体是等宽的

  • Cursive 字体

    试图模仿人的手写体

  • Fantasy 字体

    无法用任何特征定义，因为无法很容易将其划分到任何一种替他字体系列，所以单独分为一类

# 使用通用字体系列

如果你希望文档使用一种 sans-serif 字体，但是你并不关心是哪一种字体，你可以这样用：

body { font-family: sans-serif; }

只使用这些通用系列，而不做其他指定，创作人员就能创建相当复杂的样式表。

# 指定字体系列

/* 指定 Georgia 字体，没有则使用默认字体 */
h1 { font-family: Georgia; }
/* 指定 Georgia 字体，没有则使用 sans-serif 字体系列 */
h1 { font-family: Georgia, sans-serif; }

# 使用引号

如果字体名种有一个或多个空格，或者包含 # 或 $ 之类的符号，需要加引号。

单引号双引号均可。只不过如果使用双引号包围 CSS 规则时便只能使用单引号了。

# 字体加粗

关键字 100 - 900，为字体制定了 9 级加粗度，100 最细，900 最粗。

实际上，这些数字本身并没有固有的加粗度。

CSS 规范只是指出，每个数对应一个加粗度，它至少与前一个数指定的加粗度相同。

一般地，400 定义为等价于 normal，700 对应于 bold

# 让字体更粗

如果将加粗设置为 bolder，用户代理首先确定从父元素继承的 gont-weight 的值，然后选择一个属，比继承的更粗。

在满足这个条件的所有数中，要选择最小的数。

如果没有可用的字体，用户代理会把该元素加粗为一个更大的数字值，除非这个值已经是 900。

# 让字体更细

lighter，用法和 bolder 类似。

# 字体大小

font-size 的作用就是为指定字体的 em 框提供一个大小，而不能保证实际显示的字符就是这种大小。

# 绝对大小

font-size 有 7 个绝对大小值：

  • xx-small
  • x-small
  • small
  • medium
  • large
  • x-large
  • xx-large

# 相对大小

不同于加粗的相对值，相对大小值不必限制在绝对大小范围内。因此，一个字体的大小可以超过 xx-samll 和 xx-large 的大小。

# 百分数和大小

百分数值总是根据从父元素继承的大小来计算，不同于相对大小关键字，百分数允许对计算的字体大小有更细的控制。

# 字体大小与继承

font-size 是可以继承的，不过继承的是计算值而不是百分数。

# 使用长度单位

许多创作人员选择用像素值指定字体大小。

# 风格与变形

# 有风格的字体

font-style 就在 normal、italic、和 oblique 字体之间进行选择

  • normal，“非斜体或倾斜的文本”
  • italic，斜体。对每个字母的结构有一些小改动
  • bolique，倾斜，只是 normal 的倾斜版本

# 字体变形

font-variant，只有两个非继承值：normal 和 small-caps

  • small-caps：要求使用小型大写字母问题

    只要文本源中出现大写字母，会出现更大的大写字母。

    只要文本源中出现小写字母，会显示一个小型的大写字母

# font 属性

  • font-style
  • font-variant
  • font-weight
  • font-size
  • line-height
  • font-family

前三个属性可以按任何属性，但 font-size 在前，font-family 在后必须遵守

  • line-height 总在 font-size 之后

可以使用 font 作为简写

# 使用字体

如果希望一个 Web 页面结合用户的操作系统，font 的系统字体值会很方便。

  • caption，对于有标题的控件，如按钮
  • icon，对于图标加标签
  • menu，用于菜单，即下拉菜单和菜单列表
  • message-box，用于对话框
  • small-caption，用于对小控件加标签
  • status-bar，用于窗口状态条
