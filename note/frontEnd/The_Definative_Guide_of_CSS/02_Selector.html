# 选择器

规则结构:

选择器:

  • 元素选择器，最常见的选择器往往是 HTML 元素

  • 通配选择器，是一个 * ，可以与任何元素匹配

  • 类选择器，是一个（.）在类名前面

    除此之外还有多类选择器：

    <p class="urgent warning">This is a paragraph</p>

    .warning.urgent {
  font-weight: bold; /* 两个类选择器链接在一起，仅可以选择同时包含这些类名的元素*/
}

  • ID 选择器，是一个（#）在 ID 名前面

    TIP

    在 HTML 文档中，ID 选择器会使用一次且仅使用一次。

  • 简单属性选择

    h1[class] { color: silver; }	/* 选择有 class 属性的所有 h1 元素 */

  • 根据具体属性值选择

    plant[moons="1"]	/* 进一步缩小选择范围，选择有特定属性值的元素 */

    /* 把多个属性-值选择器链接在一起来选择一个文档 */
a[href="http://www.w3.org/"][title="W3C Home"] { font-size: 200%; }

  • 根据部分属性值选择

    /* 根据部分属性值选择 */
/* 效果和 .waring 一致 */
/* 但 ~= 可以用于任何属性 */
p[class~="warning"]{ font-weight: bold; }
    类型 描述
    [foo^="bar"] 选择 foo 属性值以 “bar" 开头的所有元素
    [foo$="bar"] 选择 foo 属性值以 ”bar" 结尾的所有元素
    [foo*="bar"] 选择 foo 属性值中包含子串 “bar" 的所有元素

  • 特定属性类型的选择器

    /* 选择 lang 属性等于 en 或者以 en- 开头的所有元素 */
*[lang|="en"] { color: white; }

  • 后代选择器

    TIP

    如果一个元素在另一个元素的直接上一层，它们就有父子关系。

    如果从一个元素到另一个元素的路径上要经过两层或多层，这些元素则拥有 祖先-后代关系

    h1 em { color: gray; } /* 这个规则会把作为 h1 后代的 em 元素的文本变成灰色 */

  • 子元素选择器

            p
    /   |   \
  span  a   span
        |
      strong

    p > a {} 		/* 可以 */
a > strong {}	/* 可以 */
p > strong {}	/* 不可以 */

  • 相邻兄弟结合符

    要选择紧接在另一个元素后的元素，而且二者有相同的父元素。

            div
      /     \
     ol      ul
   / | \    / | \
  li li li li li li

    ol + ul {}	/* 这就是相邻兄弟结合符 */

伪类和伪元素：

以链接为例子，以访问的链接颜色和未访问的链接颜色不一样，已访问和未访问称之为伪类，它们的选择器称之为伪类选择器。

a:visited { color: red; }	/* : 是伪类或伪元素的 "名片" */
伪类名 描述
:link 指示作为超链接（即有一个 href 属性）并指向一个未访问地址的所有锚
:visited 指示作为已访问地址超链接的所有锚
a:link { color: purple; }		/* 未访问为紫色 */
a:visited { color: silver; }	/* 已访问为银色 */

TIP

尽管 :link 和 :visited 非常有用，但它们是静态的——第一次显示之后，它们一般不会再改变文档的样式。

动态伪类：

伪类名 描述
:focus 可以接受键盘输入后者能以某种方式激活的元素
:hover 指示鼠标指针停止在哪个元素上
:active 指示被用户输入激活的元素，比如鼠标点击

选择第一个子元素：

:first-child

代码 效果

TIP

:first-child 的规则在例子中：

  • 作为某元素的第一个子元素的 p 设置为粗体
  • 作为某元素的第一个子元素的 li 变为红色

并不是 p 元素的第一个子元素，也不是 li 的第一个子元素。

根据语言选择：

*:lang(fr) { font-style: italic; } /* 把所有法语元素变为斜体 */

结合伪类：

a:link:hover { color: red; }
a:visited:hover { color: maroon; }

伪元素选择器：

为元素能够在文档中插入假想的元素，从而得到某种效果。

p:first-letter { color: red; }	/* 设置块级元素的首字母的样式 */

p:first-line { color: purple; }	/* 印象第一个文本行 */
