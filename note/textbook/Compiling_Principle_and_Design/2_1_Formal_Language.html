# 语法和语义

程序设计语言和自然语言的核心，皆由语法和语义两部分构成。语法是语言的形式，语义是语言的内容。

TIP

下面涉及了许多定义，个人没有摘抄定义，只是指出并强调可能出错的定义。

# 语法分析树与文法二义性

  • 文法二义性：如果至少存在一个句子存在两棵（或两棵以上）不同的分析树，则称该句子是二义性的。

    也可叙述为：有两个不同的最左（规范）推导，则这个文法是二义性的。

    文法的二义性不可判定，即不存在一种算法能在有限步骤内确切判断一个文法是否是二义的。

    但是可以寻找一组充分条件，使得满足这些条件的文法必定无二义性。

TIP

文法的二义性与语言语义的二义性是完全不同的概念。

并非文法有二义性，其描述的语言就有二义性，反之亦如此。

# 文法和语言的类型

乔姆斯基分类，将文法按产生式的不同分为 4 类：

  • 0 型文法（短语结构文法），可由图灵机识别
  • 1 型文法（上下文有关文法），可由线性有界自动机来识别
  • 2 型文法（上下文无关文法），可由非确定的下推自动机来识别
  • 3 型文法（正则文法、线性文法），可由有限自动机来识别

从 0 型到 3 型，后一类都是前一类的子集，且限制是逐步增强的，描述语言的功能是逐步减弱的。
