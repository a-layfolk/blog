# 像经济学家一样思考

# 作为科学家的经济学家

科学的本质是科学方法 —— 冷静地建立并检验有关世界如何运行的各种理论。

经济学家使用的科学方法如下：

  • 经济学家观察历史提供的自然实验检验理论
  • 经济学家用不同的假设来回答不同的问题
  • 经济学家使用经济模型了解世界

常用的两个模型是循环流量图和生产可能性边界

# 循环流量图

在这个模型中， 经济简单化为只由两类决策者 —— 企业和家庭 —— 所组成。 企业用劳动、 土地和资本 （建筑物和机器） 这些投入品来生产物品和服务。 这些投入品被称为生产资料。

里面的环形代表投入与产出的流向。外面的环形代表相应的货币流动。

# 生产可能性边界

生产可能性边界是一个图形， 它表明在生产要素和生产技术既定时， 一个经济所能生产的产品 —— 在这个例子中是汽车和电脑 —— 的数量的各种组合。

资源是有限的， 并不是所有组合结果均可以， 比如 C 点就不可以。

生产可能性边界上 （而不是这条线内） 的各点代表了有效率的结果。如果不减少一种物品的生产， 就没有办法生产更多的另一种物品。比如 A 点和 B 点。

D 点是一种无效率的结果。

一辆汽车的机会成本等于生产可能性边界的斜率。 在 E 点， 生产汽车的机会成本最高； 在 F 点， 生产汽车的机会成本最低。

# 微观经济学与宏观经济学

微观经济学是研究家庭和企业如何做出决策，以及它们如何在市场上相互交易的学科。

宏观经济学是研究整体经济现象， 包括通货膨胀、 失业和经济增长的学科。
