# 微型计算机硬件系统

冯·诺伊曼计算机模型硬件分为五大部分：运算器、控制器、存储器、输入设备和输出设备。

这里则将硬件划分得更加细致：主板（包括芯片组、内存、微处理器、系统总线、I/O 接口）和外部设备。

# 微处理器

# 概述

微处理器（Microprocessor）是计算机的核心，又叫“中央处理单元，即（CPU）”。是一块半导体芯片构成的电子器件。芯片上集成了算术逻辑单元（Arithmetic Login Unit，ALU）、控制单元（Control Unit，CU）和寄存器组（Registers）。

微处理器一般分为：

  • 通用微处理器
  • 专用微处理器
    • 单片机，又叫微控制器（Micro Controller Unit，MCU），常用于控制领域
    • 数字信号处理器。主要用在通信、消费类电子产品领域。

通用微处理器和专用微处理器基本原理大同小异、技术上相同，应用在不同的领域。

# 80x86 系列 CPU 的发展

  • 4 位微处理器，直到今天仍应用于一些嵌入式系统，比如微波炉、洗衣机、计算器
  • 8 位微处理器
  • 16 位微处理器
  • 32 位微处理器
  • 64 位微处理器

在 下x86 的 16 位和 32 位时代，技术上都是 Intel 公司来主导，AMD 与之兼容。而在 x86-64 上，部分技术的主导权已经属于 AMD。

# CPU 的微结构

微结构（Micro-Architecture）也叫计算机组织，包含处理器内部的构成及这些构成的部分如何执行指令集。

# 微处理器性能指标

# 速度指标

  • 主频，也叫主时钟频率，表示 CPU 内数字脉冲信号振荡频率。主频越高，一个时钟周期内完成的指令数越多，CPU 运算速度也越快。
  • 外频，是系统总线的工作频率，即 CPU 与周边设备传输数据的频率。可以理解为外频直接影响内存的访问速度，外聘越高，速度越快。
  • 倍频：主频 = 外频 × 倍频

课本中提到了北桥和前端总线，但近年来内存控制器被逐步集成到处理器中，CPU 直接和内存通信，不再通过北桥，前端总线已经消失了。

# 高速缓冲存储器

为了协调 CPU 主频和内存总线频率之间的差异，计算机设置了高速缓冲存储器（Cache）。

CPU 一般有一级缓存（L1 Cache）和二级缓存（L2 Cache），高端的有三级缓存（L3 Cache）。

一级缓存常采用哈佛结构，分为数据缓存和指令缓存。速度上 ：L1 > L2 > L3

# 制造工艺

数值线宽是 IC 生产工艺可达到的最小导线长度。线宽越小，意味着相同面积下可以集成更多的晶体管，实现更复杂的功能和更高的性能。

# 核心电压

CPU 的工作电压（Supply Voltage），即 CPU 正常工作所需的电压，包括：

  • CPU 的核心电压
  • I/O 电压

通常 CPU 的核心电压 <= I/O 电压

降低电压是 CPU 主频提高的重要因素之一，较低的工作带你呀能够降低 CPU 的公安后，减少发热量。

# 封装形式

CPU 封装是采用特定的材料将 CPU 芯片或 CPU 模块固化在其中以仿损坏的保护措施。

# 指令系统

分为：

  • 复杂指令集（Complex Instruction Set Computer，CISC）
  • 精简指令集（Reduced Instruction Set Computer，RISC）

# 超线程技术

理论上，实行超线程技术（Hyper-Threading Technology）之后，一个物理处理器核上会模拟出两个逻辑内核，每一个内核模拟成一个 CPU 芯片，实现线程级别上并行处理。

# 超标量核超长指令字

  • 超标量（Super Scalar）技术指的是 CPU 在同一时刻执行两条或两条以上指令的能力。
  • 超长指令字（Very Long Instruction Word，VLIW）使用多个相同功能部件执行一条超长指令，实现指令级的并行处理，从而提高性能
# 动态执行技术

动态执行使 CPU 通过更符合逻辑的顺序而不是简单地按指令序列来执行，从而获得更高的效率。

# 多核技术

单核处理器面临的困境：

  • 功耗。晶体管通过翻转来提供信息计算，翻转的同时会产生热量。
  • 互连线延迟。晶体管速度急剧增长，但是互连线延迟却没有大的变化。
  • 设计复杂度。

多核是指在一个处理器中集成两个或多个完成的计算引擎（内核）。多核之难不在核上，而在互连与编程两大挑战上。

# 主板

# 主板结构

主板（Main Board）是微型机各种硬件的载体，主要功能是管理和协调微机系统的各部件。

比较通用的主板结构主要包括 AT、ATX 和 BTX 三大类。

  • AT 结构：主要在早期的微型计算机中使用，目前已经被淘汰。
  • ATX 结构：得到世界主要厂商的支持，是目前最广泛的工业标准。
  • BTX 结构：诞生初期发展势头很好，但由于与 ATX 的兼容问题和产品换代成本过高，于 2006 年被 Intel 放弃。

# 芯片组

芯片组（Chipser）是主板的核心组成部分，主要包括：

  • 控制芯片组

    • 南北桥体系结构：CPU + 南桥 + 北桥

      • 北桥，内部集成内存控制器，故称内存控制器集线器（Memory Controller Hub，MCH），负责对内存的读写控制和数据传输

      • 南桥，又称 I/O 控制中心（I/O Controller Hub，ICH）。负责对 I/O 设备进行读写控制和数据的传输处理，提供各种外设的接口。

    • 单芯片组体系结构：北桥被取消，只剩 CPU + 南桥

  • BIOS 芯片：（Basic Input/Output System），只读存储器基本输入、输出系统。记录系统一些基本设置信息，比如系统设置信息，

  • CMOS 芯片：可读写的 RAM 芯片，保存系统的硬件配置。现在厂商把 CMOS 程序固化到 BIOS 芯片中。

存储器访问

内存最基本的存储单位是字节，内存的物理地址也是以字节为单位的。

  • 在访问字节数据时，直接给出所在的地址号即可

  • 对于字、双字、四字数据类型，Intel CPU 在访问这些数据时，只需给出最低单元的地址号，然后依次存储后续字节即可。

    Intel CPU 采用低字节在前（Little Endian）的存储格式，即低地址中存放低字节地址，高地址中存放高字节地址
