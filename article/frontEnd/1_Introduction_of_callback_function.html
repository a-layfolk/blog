# 浅谈回调函数

回调函数是一个作为参数传递的函数。A 函数调用 B 函数时，B 函数却要求 A 函数先给自己传入一个 C 函数。C 函数就是回调函数。

举网购商品为例子，你告诉商家你要购买的商品，以及邮寄给你的方式。你相当于 A 函数，商家相当于 B 函数，邮寄方式相当于 C 函数。

商家打包商品的流程是固定的，即 B 函数流程是固定的。但根据你留下的邮寄方式的不同，商家可能会邮寄到北京，或者邮寄到上海，可能使用快递、空邮或海邮，即 C 函数会决定 B 函数之后的流程。

我最早接触的回调函数是 C 语言的快排函数 qsort, 它使用函数指针来实现回调函数

void qsort(void *base, size_t nitems, size_t size, int (*compar)(const void *, const void*))

目前接触得最多的是 NodeJS 中的函数，它使用匿名函数

var fs = require('fs');
fs.readFile('./text.txt', function (err, data) {
  if(err) {
    console.log('读取文件出错');
  }
  else {
    console.log(data.toString());
  }
})
